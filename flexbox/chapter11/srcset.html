Srcset Test Results

Here are the results of some experiments with the srcset attribute and flexbox.

  • Image displayed. Mozilla Firefox displays different images as screen shrinks and expands. Chrome and Opera display initial image based on viewport size. As viewport grows, it displays bigger images. Once a bigger image is displayed, that image continues to be displayed even when the viewport shrinks. MS Edge behaves similarly; except that sometimes when the cat image is supposed to be displayed on small viewports (/images/kitten-568.jpg 400w,), edge doesn't display any image. When the dog image is supposed to be displayed on small viewports (/images/dog-865px.jpg 400w,), it does get displayed.
  • Srcset. Srcset works on all modern browsers, except IE11. IE displays the "src" images.
  • Center flex container. Cannot center flex container with "margin: 20px auto;" on Mozilla Firefox.
  • The "flex:none;" declaration. The image inside the flex container does not need a "flex:none;" declaration to maintain its proportions as it shrinks and expands.
