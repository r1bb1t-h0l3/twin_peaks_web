Sizes Test Results

Here are the results of some experiments with the sizes attribute and flexbox.

  • Image displayed. Mozilla Firefox displays different images as screen shrinks and expands. Chrome had problems. Sometimes, displayed big image when it was too large for screen. Opera initially displayed small images; but once big image (elephant) appeared, it only showed elephant at 50% viewport width. IE11 showed lemur at all screen widths. Edge did not display any image when the page loaded intially at a narrow width.
